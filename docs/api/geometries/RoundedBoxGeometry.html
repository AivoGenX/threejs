[name]

Version of box geometry with rounded edges.

Example

var geometry = new THREE.RoundedBoxGeometry( 10, 10, 10, 2, 3 ); var material = new THREE.MeshBasicMaterial( {color: 0x00ff00} ); var cube = new THREE.Mesh( geometry, material ); scene.add( cube );

Constructor

[name]([page:Float width], [page:Float height], [page:Float depth], [page:Float radius], [page:Integer radiusSegments])

width — Width of the sides on the X axis.
height — Height of the sides on the Y axis.
depth — Depth of the sides on the Z axis.
radius — Radius of edge fillet.
radiusSegments — Optional. Number of segmented faces along the edges of the box. Default 1.

Properties

.parameters

Using the above example code above as our basis:

geometry.parameters; // outputs an object {width: 1, height: 1, depth: 1, radius: .25, radiusSegments: 1} cube.geometry.parameters; // as above cube.geometry.parameters.width; // === 1 cube.geometry.parameters.radiusSegments // === 1.
