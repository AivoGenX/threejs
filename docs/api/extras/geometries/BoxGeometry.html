[name]

BoxGeometry is the quadrilateral primitive geometry class. It is typically used for creating a cube or cuboid of the dimensions provided with the 'width', 'height', and 'depth' constructor arguments.

Example

var material = new THREE.MeshBasicMaterial({ color: 0x0000ff }); var width = 5, height = 5, depth = 5; var boxGeometry = new THREE.BoxGeometry( width, height, depth ); var box = new THREE.Mesh( boxGeometry, material ); scene.add( box );

Constructor

[name]([page:Float width], [page:Float height], [page:Float depth], [page:Integer widthSegments], [page:Integer heightSegments], [page:Integer depthSegments])

width - width of the box height - height of the box depth - depth of the box widthSegments - number of width-wise segments heightSegments - number of height-wise segments depthSegments - number of depth-wise segments
