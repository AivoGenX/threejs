[name]

A helper class to allow textures referenced in model files to be loaded from a different URL.

Properties

.[page:string texturePath]

The path that will be prepended to file names.

.[page:function defaultGetter]

The function that will be called when no registry entry is present for a file. The default behavior is to prepend the texturePath.

Methods

.get([page:string sourceFile]) [page:string]

sourceFile -- The file name to fetch from the registry.
Will return the entry for the given source file if one is present, or call the default getter.

.put([page:string sourceFile], [page:function getter]) [page:string]

sourceFile -- The file name to enter into the registry.
getter -- The function to call when this file is requested.
Will register the given getter function to be called when the same sourceFile is passed to get. If the given getter is a string, that string will be passed to the default getter, allowing simple aliasing of file names.
