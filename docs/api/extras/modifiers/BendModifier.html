[name]

The Bend modifier lets you bend the current selection up to 90 degrees about a single axis, producing a uniform bend in an object's geometry.

Example

var geometry = new THREE.BoxGeometry( 1500, 800, 300, 200, 10, 4 ); var material = new THREE.MeshLambertMaterial( {color: 0x00ff00} ); var Cube = new THREE.Mesh( geometry, material ); scene.add( Cube ); var modifier = new THREE.BendModifier(); var direction = new THREE.Vector3( 0, 0, -1 ); var axis = new THREE.Vector3( 1, 0, 0 ); var angle = Math.PI / 6; modifier.set( direction, axis, angle ).modify( Cube );
