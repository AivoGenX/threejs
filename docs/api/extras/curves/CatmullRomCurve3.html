[name]

Create a smooth 3d spline curve from a series of points using the [link:https://en.wikipedia.org/wiki/Centripetal_Catmull-Rom_spline Catmull-Rom] algorithm.

Example

//Create a closed wavey loop var curve = new THREE.CatmullRomCurve3( [ new THREE.Vector3( -10, 0, 10 ), new THREE.Vector3( -5, 5, 5 ), new THREE.Vector3( 0, 0, 0 ), new THREE.Vector3( 5, -5, 5 ), new THREE.Vector3( 10, 0, 10 ) ] ); var geometry = new THREE.Geometry(); geometry.vertices = curve.getPoints( 50 ); var material = new THREE.LineBasicMaterial( { color : 0xff0000 } ); // Create the final object to add to the scene var curveObject = new THREE.Line( geometry, material );

Constructor

[name]( [page:Array points] )

Properties

See the base [page:Curve] class for common properties.

[property:Array points]

The array of array of [page:Vector3] points that define the curve.

[property:Boolean closed]

The curve will loop back onto itself when this is true. False by default

[property:String type]

Possible values are `centripetal` (default), `chordal` and `catmullrom`.

[property:float tension]

When [page:.type] is `catmullrom`, defines catmullrom's tension. Defaults is *0.5*.

[property:Array hermiteSegments]

The computed hermite segments from the given points.

[property:Array timeValues]

The time at which each given point will be crossed.

[property:Boolean segmentsOutdate]

If the points are changed in any way, this must be set to true to recalculate the segments.

Methods

See the base [page:Curve] class for common methods.

[method:void generateSegments]()

Generate and cache Hermite segments from the points

[method:Object getCatmullSegment]( [page:Float t] )

Given t, greater than 0, grab the segment that contains the value. Each segment covers a different span of time depending on distance between points.

[method:Vector getCatmullPoint]( [page:Float t] )

Given t, greater than 0, grab the point for the given time treating the points as separated by their physical distance.

[method:Object getNormalizedSegment]( [page:Float t] )

Given t, in the range ( 0 .. 1 ), returns the segment that contains the point. Each segment covers the same span of time.

[method:Vector getNormalizedPoint]( [page:Float t] )

Given t, in the range ( 0 .. 1 ), returns the matching point on the curve treating the points as evenly separated.
