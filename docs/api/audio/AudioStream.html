[name]

Wrapper for a HTML5 audio stream.

Example

[example:misc_sound_stream misc / sound_stream ]
// create an AudioListener and add it to the camera var listener = new THREE.AudioListener(); camera.add( listener ); // create an audio stream var audioStream = new THREE.AudioStream( 'myaudio.mp3', listener ); audioStream.play();

Constructor

[name]( [page:String path], [page:AudioListener listener] )

path — (required) [page:String path] to the audio file.
listener — (required) [page:AudioListener AudioListener] instance.

Properties

[property:HTMLAudioElement audioElement]

The [link:https://developer.mozilla.org/de/docs/Web/API/HTMLAudioElement HTMLAudioElement] that represents the actual stream.

[property:AudioContext context]

The [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext AudioContext] of the [page:AudioListener listener] given in the constructor.

[property:GainNode gain]

A [link:https://developer.mozilla.org/en-US/docs/Web/API/GainNode GainNode] created using [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext/createGain AudioContext.createGain]().

[property:String source]

An [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioBufferSourceNode AudioBufferSourceNode] created using [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioContext/createBufferSource AudioContext.createBufferSource]().

Methods

[method:GainNode getOutput]()

Returns the [page:.gain gainNode].

[method:null play]()

Starts playback.

[method:null pause]()

Pauses playback.

[method:Number getCurrentTime]()

Returns the current time of the audio.

[method:Number setCurrentTime]( value )

Sets the current time of the audio.

[method:Number getVolume]()

Returns the current volume.

[method:Number setVolume]( value )

Sets the current volume.

[method:Number getLoop]()

Returns the loop property of the [page:.audioElement].

[method:Number setLoop]( value )

Sets the loop property of the [page:.audioElement]. If set to true, the audio will start over again, every time it is finished.

[method:null connect]()

Connects to the output node.

[method:null disconnect]()

Disconnects from the output node.
