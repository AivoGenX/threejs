Uniforms library for shared webgl shaders

A Note on Usage

These methods perform deep copies of the uniform definitions. Attempting to clone or merge uniforms with a render target texture is an invalid operation. The following code will fail: const cubeRenderTarget = new THREE.WebGLCubeRenderTarget(128); const uniforms = THREE.UniformsUtils.merge([ {}, { cubemap: { value: cubeRenderTarget.texture }, }]); Instead, you are advised to set any render target textures AFTER merging or cloning uniforms: const cubeRenderTarget = new THREE.WebGLCubeRenderTarget(128); const uniforms = THREE.UniformsUtils.merge([ {}, { cubemap: { value: null }, }]); uniforms.cubemap.value = cubeRenderTarget.texture;
