Ce guide fournit un bref aperçu des composants de base d'une application de réalité virtuelle basée sur le Web. fait avec three.js.

Flux de travail

Tout d'abord, vous devez inclure [link: https://github.com/mrdoob/three.js/blob/master/examples/jsm/webxr/VRButton.js VRButton.js] dans votre projet.

import { VRButton } from 'three/examples/jsm/webxr/VRButton.js';

*VRButton.createButton()* fait deux choses importantes : Il crée un bouton qui indique Compatibilité RV. De plus, il lance une session VR si l'utilisateur active le bouton. La seule chose qu'il vous reste as à faire est d'ajouter la ligne de code suivante à votre application.

document.body.appendChild( VRButton.createButton( renderer ) );

Ensuite, vous devez indiquer à votre instance de *WebGLRenderer* pour activer le rendu XR.

renderer.xr.enabled = true;

Enfin, vous devez ajuster votre boucle d'animation car nous ne pouvons pas utiliser notre bien connu Fonction *window.requestAnimationFrame()*. Pour les projets VR, nous utilisons [page:WebGLRenderer.setAnimationLoop setAnimationLoop]. Le code minimal ressemble à ceci :

renderer.setAnimationLoop( function () { renderer.render( scene, camera ); });
