Même si cela devient de moins en moins un problème, certains appareils ou navigateurs peuvent toujours ne pas prendre en charge WebGL. La méthode suivante vous permet de vérifier si elle est prise en charge et d'afficher un message à l'utilisateur si ce n'est pas le cas.

Ajouter [lien : https://github.com/mrdoob/three.js/blob/master/examples/jsm/WebGL.js] à votre javascript et exécutez ce qui suit avant d'essayer de rendre quoi que ce soit.

if ( WEBGL.isWebGLAvailable() ) { // Lancer la fonction ou d'autres initialisations ici animate(); } else { const warning = WEBGL.getWebGLErrorMessage(); document.getElementById( 'container' ).appendChild( warning ); }
