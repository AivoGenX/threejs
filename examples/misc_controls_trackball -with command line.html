three.js - trackball controls example
MOVE mouse & press LEFT/A: rotate, MIDDLE/S: zoom, RIGHT/D: pan
This test that the THREE.TrackballControls processes the key-event correctly and do not propagate key event when the controlled domElement ( parent of the canvas) has focus.
  • set the focus to one input field below
  • press the left mouse button inside the WebGl view and start twiddling about to rotate the view
  • while moving the mouse with the left button pressed, press the "D" key to pan the view
  • verify that the input field are not garbaged with unwanted characters.
